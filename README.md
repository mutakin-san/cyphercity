# Cyphercity

Competition Turnament Apps. Built with [Flutter](https://flutter.dev/)



## Features
- Login & Register
- Register Event
- Management Team
- Highlights
- Event Update
- Multi User


## Tech Stack

- Flutter
- BLoC (State Management)
- Shared Preferences
- HTTP Networking


## Installation


```bash
  git clone https://github.com/mutakin-san/cyphercity.git
  flutter pub get
  flutter run 
```
